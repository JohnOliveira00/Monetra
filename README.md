# Monetra Landing Page

---

##  Sobre o Projeto 

O **Monetra Landing** é uma landing page desenvolvida com foco em apresentar uma solução digital voltada para gestão financeira de forma moderna, intuitiva e responsiva.

O projeto tem como objetivo demonstrar habilidades em desenvolvimento front-end, criando uma interface atrativa que comunica valor ao usuário e melhora a experiência de navegação.

---

##  Objetivos

- Criar uma landing page moderna e responsiva  
- Aplicar boas práticas de UI/UX  
- Estruturar um layout limpo e profissional  
- Simular uma apresentação de produto digital (finanças)
- Com interatividades que atraem o usuário


---

##  Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript
- Supabase


---

##  Funcionalidades

- Organização financeira sem a necessidade de anotações ou planilhas complexas
- Foco em bater metas financeiras e economizar com mais facilidade
- Layout totalmente responsivo  
-  Design moderno e intuitivo  
-  Navegação fluida  
-  Estrutura organizada e escalável
